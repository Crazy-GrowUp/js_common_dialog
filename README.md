# 项目介绍
此项目主要是以javascript为基础，编写的通用弹框组件。
其他包含了六种弹框 1.普通弹框 2.带输入框的弹框 3.带输入框的文件选择弹框 4.进度条弹框 5.带组合选择的弹框 6.加载旋转框
